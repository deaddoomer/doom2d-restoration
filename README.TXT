This branch contains restored sources of Doom2D v1.40
Based on original sources of Doom2D v1.35 published by Prikol Software

How to build:
- Put WSTUB.EXE from Watcom C/C++ 10.0 (exact this version) into directory GAME
- Install Watcom C/C++ 11.0 (exact this version without patches A/B/C)
- Install Borland C++ 3.1 (or Borland Pascal 7.0)
- Setup all required environment variables (%PATH% %INCLUDE% %WATCOM% %EDPATH%)
- Execute BUILD.BAT
- DOOM2D.EXE will appear in directory GAME
