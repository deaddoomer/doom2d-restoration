This branch contains restored sources of Doom2D v1.30
Based on original sources of Doom2D v1.35 published by Prikol Software

How to build:
- Put PMODEW.LNK & PMODEW.EXE into directory GAME (PMODE/W v1.16)
- Install Watcom C/C++ 10.0 (exact this version without patches 10.0a/10.0b)
- Install Borland C++ 3.1
- Setup all required environment variables (%PATH% %INCLUDE% %WATCOM% %EDPATH%)
- Execute BUILD.BAT
- DOOM2D.EXE will appear in directory GAME
