This branch contains restored sources of Doom2D v1.35
Based on original sources of Doom2D v1.35 published by Prikol Software

How to build:
- Install Watcom C/C++ 10.x (Watcom C/C++ 10.0 recommended)
- Install Borland C++ 3.x (Borland C++ 3.1 recommended)
- Setup all required environment variables (%PATH% %INCLUDE% %WATCOM% %EDPATH%)
- Execute BUILD.BAT
- DOOM2D.EXE will appear in directory GAME
